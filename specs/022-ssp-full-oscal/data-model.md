## New Entities

### SspSection

Stores individual SSP sections with lifecycle tracking. One record per (system, section number). Auto-generated sections have content populated from entity data; authored sections contain ISSO-written markdown.

**Location**: `src/Ato.Copilot.Core/Models/Compliance/RmfModels.cs`

```csharp
/// <summary>Individual NIST SP 800-18 SSP section with lifecycle tracking.</summary>
public class SspSection
{
    /// <summary>Unique identifier (GUID).</summary>
    [Key]
    [MaxLength(36)]
    public string Id { get; set; } = Guid.NewGuid().ToString();

    /// <summary>System this section belongs to.</summary>
    [Required]
    [MaxLength(36)]
    public string RegisteredSystemId { get; set; } = string.Empty;

    /// <summary>NIST 800-18 section number (1–13).</summary>
    [Required]
    [Range(1, 13)]
    public int SectionNumber { get; set; }

    /// <summary>Section title (e.g., "System Identification").</summary>
    [Required]
    [MaxLength(200)]
    public string SectionTitle { get; set; } = string.Empty;

    /// <summary>Section content (markdown format).</summary>
    [MaxLength(32000)]
    public string? Content { get; set; }

    /// <summary>Section lifecycle status.</summary>
    [Required]
    public SspSectionStatus Status { get; set; } = SspSectionStatus.NotStarted;

    /// <summary>Whether this section is auto-generated from entity data.</summary>
    public bool IsAutoGenerated { get; set; }

    /// <summary>Whether an auto-generated section has been manually overridden.</summary>
    public bool HasManualOverride { get; set; }

    /// <summary>User who authored or last modified the section.</summary>
    [Required]
    [MaxLength(200)]
    public string AuthoredBy { get; set; } = string.Empty;

    /// <summary>When the section was last authored/modified (UTC).</summary>
    public DateTime AuthoredAt { get; set; } = DateTime.UtcNow;

    /// <summary>Reviewer who approved or rejected the section.</summary>
    [MaxLength(200)]
    public string? ReviewedBy { get; set; }

    /// <summary>When the section was reviewed (UTC).</summary>
    public DateTime? ReviewedAt { get; set; }

    /// <summary>Reviewer comments (populated on rejection).</summary>
    [MaxLength(4000)]
    public string? ReviewerComments { get; set; }

    /// <summary>Optimistic concurrency version (auto-incremented on update).</summary>
    [ConcurrencyCheck]
    public int Version { get; set; } = 1;

    // ─── Navigation property
    /// <summary>Parent system.</summary>
    public RegisteredSystem? RegisteredSystem { get; set; }
}
```

**Unique constraint**: `(RegisteredSystemId, SectionNumber)` — only one record per section per system.

**Lifecycle rules**:
- Status transitions: NotStarted → Draft → UnderReview → Approved (strict sequential)
- Auto-generated sections (§1,§2,§3,§4,§7,§9,§10,§11) start at Draft
- Authored sections (§5,§8,§12,§13) start at NotStarted
- Hybrid section (§6) starts at NotStarted
- Content update on Approved section resets status to Draft
- `Version` is an EF Core concurrency token — writes require expected version match

### ContingencyPlanReference

Stores metadata for an externally-authored contingency plan document referenced in SSP §13. Follows the same document-reference pattern as `InterconnectionAgreement`.

**Location**: `src/Ato.Copilot.Core/Models/Compliance/RmfModels.cs`

```csharp
/// <summary>Reference to an external contingency plan document for SSP §13.</summary>
public class ContingencyPlanReference
{
    /// <summary>Unique identifier (GUID).</summary>
    [Key]
    [MaxLength(36)]
    public string Id { get; set; } = Guid.NewGuid().ToString();

    /// <summary>System this contingency plan belongs to.</summary>
    [Required]
    [MaxLength(36)]
    public string RegisteredSystemId { get; set; } = string.Empty;

    /// <summary>Document title (e.g., "ACME IT Contingency Plan v2.1").</summary>
    [Required]
    [MaxLength(500)]
    public string DocumentTitle { get; set; } = string.Empty;

    /// <summary>Document location (URL, file path, or SharePoint reference).</summary>
    [Required]
    [MaxLength(1000)]
    public string DocumentLocation { get; set; } = string.Empty;

    /// <summary>Document version string.</summary>
    [MaxLength(50)]
    public string? DocumentVersion { get; set; }

    /// <summary>When the contingency plan was last tested (UTC).</summary>
    public DateTime? LastTestedDate { get; set; }

    /// <summary>Type of last test (tabletop, functional, full-scale).</summary>
    [MaxLength(50)]
    public string? TestType { get; set; }

    /// <summary>Recovery Time Objective (e.g., "4 hours").</summary>
    [MaxLength(100)]
    public string? RecoveryTimeObjective { get; set; }

    /// <summary>Recovery Point Objective (e.g., "1 hour").</summary>
    [MaxLength(100)]
    public string? RecoveryPointObjective { get; set; }

    /// <summary>Alternate processing site location.</summary>
    [MaxLength(500)]
    public string? AlternateProcessingSite { get; set; }

    /// <summary>Summary of backup procedures.</summary>
    [MaxLength(4000)]
    public string? BackupProceduresSummary { get; set; }

    /// <summary>User who created the reference.</summary>
    [Required]
    [MaxLength(200)]
    public string CreatedBy { get; set; } = string.Empty;

    /// <summary>Creation timestamp (UTC).</summary>
    public DateTime CreatedAt { get; set; } = DateTime.UtcNow;

    /// <summary>Last modification (UTC).</summary>
    public DateTime? ModifiedAt { get; set; }

    // ─── Navigation property
    /// <summary>Parent system.</summary>
    public RegisteredSystem? RegisteredSystem { get; set; }
}
```

### OnModelCreating Configuration

```csharp
// ── SspSection
modelBuilder.Entity<SspSection>(entity =>
{
    entity.HasKey(e => e.Id);
    entity.Property(e => e.Id).HasMaxLength(36);
    entity.Property(e => e.RegisteredSystemId).HasMaxLength(36).IsRequired();
    entity.Property(e => e.SectionTitle).HasMaxLength(200).IsRequired();
    entity.Property(e => e.Content).HasMaxLength(32000);
    entity.Property(e => e.AuthoredBy).HasMaxLength(200).IsRequired();
    entity.Property(e => e.ReviewedBy).HasMaxLength(200);
    entity.Property(e => e.ReviewerComments).HasMaxLength(4000);

    // Enum conversion to string
    entity.Property(e => e.Status)
        .HasConversion<string>()
        .HasMaxLength(20);

    // Concurrency token
    entity.Property(e => e.Version).IsConcurrencyToken();

    // Unique constraint: one section per system per section number
    entity.HasIndex(e => new { e.RegisteredSystemId, e.SectionNumber })
        .IsUnique()
        .HasDatabaseName("IX_SspSection_System_Number");

    entity.HasIndex(e => e.Status)
        .HasDatabaseName("IX_SspSection_Status");
});

// ── RegisteredSystem → SspSections (one-to-many, CASCADE delete)
// Add to existing RegisteredSystem configuration block:
entity.HasMany(e => e.SspSections)
    .WithOne(s => s.RegisteredSystem)
    .HasForeignKey(s => s.RegisteredSystemId)
    .OnDelete(DeleteBehavior.Cascade);

// ── RegisteredSystem → ContingencyPlanReference (one-to-one, CASCADE delete)
entity.HasOne(e => e.ContingencyPlanReference)
    .WithOne(c => c.RegisteredSystem)
    .HasForeignKey<ContingencyPlanReference>(c => c.RegisteredSystemId)
    .OnDelete(DeleteBehavior.Cascade);

// ── RegisteredSystem — new enum conversion
entity.Property(e => e.OperationalStatus)
    .HasConversion<string>()
    .HasMaxLength(20);

// ── ContingencyPlanReference
modelBuilder.Entity<ContingencyPlanReference>(entity =>
{
    entity.HasKey(e => e.Id);
    entity.Property(e => e.Id).HasMaxLength(36);
    entity.Property(e => e.RegisteredSystemId).HasMaxLength(36).IsRequired();
    entity.Property(e => e.DocumentTitle).HasMaxLength(500).IsRequired();
    entity.Property(e => e.DocumentLocation).HasMaxLength(1000).IsRequired();
    entity.Property(e => e.DocumentVersion).HasMaxLength(50);
    entity.Property(e => e.TestType).HasMaxLength(50);
    entity.Property(e => e.RecoveryTimeObjective).HasMaxLength(100);
    entity.Property(e => e.RecoveryPointObjective).HasMaxLength(100);
    entity.Property(e => e.AlternateProcessingSite).HasMaxLength(500);
    entity.Property(e => e.BackupProceduresSummary).HasMaxLength(4000);
    entity.Property(e => e.CreatedBy).HasMaxLength(200).IsRequired();

    // Unique constraint: one per system
    entity.HasIndex(e => e.RegisteredSystemId)
        .IsUnique()
        .HasDatabaseName("IX_ContingencyPlan_SystemId");
});
```

## NIST 800-18 Section → Entity Data Source Map

| § | Section | Type | Primary Data Sources |
|---|---------|------|---------------------|
| 1 | System Identification | Auto | RegisteredSystem (Name, Acronym, DitprId, EmassId, Description, SystemType, MissionCriticality) |
| 2 | Security Categorization | Auto | SecurityCategorization, InformationType (FIPS 199 levels) |
| 3 | System Owner / AO | Auto | RmfRoleAssignment (filtered by RmfRole: SystemOwner, AuthorizingOfficial, Isso, Issm) |
| 4 | System Type | Auto | RegisteredSystem (SystemType, MissionCriticality, IsNationalSecuritySystem) |
| 5 | General Description / Purpose | Authored | SspSection.Content (markdown), RegisteredSystem.Description as template |
| 6 | System Environment | Hybrid | RegisteredSystem.HostingEnvironment + AzureProfile (auto) + SspSection.Content (authored narrative) |
| 7 | Interconnections | Auto | SystemInterconnection, InterconnectionAgreement (Feature 021) |
| 8 | Laws / Regulations | Authored | SspSection.Content (markdown) |
| 9 | Minimum Security Controls | Auto | ControlBaseline, ControlTailoring, ControlImplementation (summary counts + tailoring rationale) |
| 10 | Control Implementations | Auto | ControlImplementation (per-control narratives grouped by family) |
| 11 | Authorization Boundary | Auto | AuthorizationBoundary (resource inventory), SspSection.Content (narrative overlay) |
| 12 | Personnel Security | Authored | SspSection.Content (markdown), RmfRoleAssignment (auto-populated role list) |
| 13 | Contingency Plan | Authored | SspSection.Content (markdown), ContingencyPlanReference (structured metadata) |
