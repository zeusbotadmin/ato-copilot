## Story

An authenticated user clicks "Sign out" from the top-right account
menu. The session is invalidated server-side, the local SPA state is
cleared (Redux / React Query caches, localStorage conversation history
from 034-dashboard-chat, and the `ato-impersonate` cookie from
048-tenant-isolation if present), and the user is redirected to
`/login` with a confirmation toast. The same full-sign-out path runs
automatically when the user has been idle past `Auth:IdleTimeoutMinutes`
(default 30).

## Acceptance scenarios

1. **Given** an authenticated session, **When** the user clicks "Sign out", **Then** the SPA calls `POST /api/auth/signout`, the server clears first-party cookies + the `ato-impersonate` cookie, and the user is redirected to `/login`.
2. **Given** sign-out during an active CSP-Admin impersonation, **When** sign-out completes, **Then** the `ato-impersonate` cookie is deleted in addition to the auth token.
3. **Given** the dashboard chat panel is open, **When** sign-out completes, **Then** in-memory chat state is cleared, but localStorage history remains tied to the user's `oid`.
4. **Given** a session has been idle past `Auth:IdleTimeoutMinutes`, **When** the timer fires, **Then** the SPA performs a **full sign-out** — revoke server session, clear local state, redirect to `/login?reason=idle_timeout`. No overlay, no silent renew.
5. **Given** the user is mid-edit on a form when idle expires, **When** the timer fires, **Then** the SPA captures the in-flight form state to localStorage BEFORE sign-out so a "Restore unsaved changes" prompt can offer recovery on next sign-in.

## Functional requirements covered

- **FR-005**: `POST /api/auth/signout` tears down session + returns 204.
- **FR-006**: SPA clears Redux / React Query caches, in-memory chat state, and the impersonation cookie.
- **FR-007**: Idle timer triggers the same full-sign-out path as explicit sign-out; redirect `/login?reason=idle_timeout`.
- **FR-007a**: Access-token renewal owned by MSAL.js silent renewal; the idle timer counts user input, not token refresh.
- **FR-008**: Persist in-flight form state to localStorage namespaced by `oid` before idle sign-out.
