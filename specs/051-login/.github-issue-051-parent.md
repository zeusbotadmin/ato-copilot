# Feature 051 — First-Class Login Experience

## Summary

ATO Copilot has authentication plumbing today — CAC/PIV via MSAL
([003-cac-auth-pim](../specs/003-cac-auth-pim/)), simulation mode for
dev ([027-cac-simulation-mode](../specs/027-cac-simulation-mode/)),
tenant isolation + CSP-Admin impersonation
([048-tenant-isolation](../specs/048-tenant-isolation/)) — but no
first-class **Login** experience for the Dashboard, VS Code extension,
M365 / Teams app, or Web Chat surface. Users landed on protected pages
and the browser silently negotiated Entra.

Feature 051 ships the missing surface across all four channels: a
branded `/login` page, a tenant / organisation picker, idle-timeout
sign-out, per-class error pages, a developer simulation panel, a
CSP-Admin impersonation banner, a full account menu, and an immutable
13-month audit trail (`LoginAuditEvents`) with cold-archive overflow.
